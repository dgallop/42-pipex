## Pipex

This poject recreates the use of pipes such as:
```
<infile command1 | command2 > outfile
```
How to use the program after being compiled:
```
./pipex infile command1 command2 outfile
```
```
./pipex hello.txt "ls -l" "wc -l" bye.txt
```
